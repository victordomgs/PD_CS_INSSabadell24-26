# Actividad 3: Suma Binaria

## Reglas de la Suma Binaria

1. **0 + 0 = 0**
2. **0 + 1 = 1**
3. **1 + 0 = 1**
4. **1 + 1 = 10** (escribimos 0 y llevamos 1 al siguiente dígito, como en el sistema decimal)

## Ejemplo

**Suma los números binarios siguientes:**

1. 1011 + 1101
2. 10101 + 111
3. 1110 + 1010
4. 10011 + 1101
5. 11001 + 11001
6. 1011 + 1111
7. 111111 + 101
8. 100101 + 11101
9. 110011 + 1101
10. 1001 + 110
